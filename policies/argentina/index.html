    • Spanish: Repositorio digital (español)

Accesibilidad de la Información en las Páginas Web

Official law/policy page

  • Spanish: Accesibilidad de la Información en las Páginas Web (español)

Details

  • Date enacted: 2010
  • Responsible Entity (Agency, Ministry, etc.):
    • Spanish: Oficina Nacional de Tecnologías de Información (español)
  • Type: Procurement law
  • Scope:
  • Web only: yes
  • WCAG Version Used: WCAG 2.0
  • Relevant Standard:
    • WCAG 2.0
      Conformance level A
  • Relevant Documents:
    • Spanish: Web Accessibility Repository (español)
